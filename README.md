# Transfer-Learning-for-GDE-Identification
Transfer Learning for GDE Identification
We have open-sourced our transfer learning implementation on GitHub to support reproducibility. 
The compiled datasets are not included in the repository due to file size limits, 
but we would be glad to share them directly with anyone interested—
all source data are drawn from publicly available resources, and no proprietary restrictions apply. 
The complete Google Earth Engine collection scripts and remaining auxiliary codes will be added as soon as the paper reaches the revision or acceptance stage.
We will prepare environment setup documentation in follow-up work.
